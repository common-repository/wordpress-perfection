This plugin will help you to install all wordpress "must have" plugins just in one click.
   

== Description ==

If you just installed wordpress and searching for plugins to make your wordpress site complete then you search ends here. As this plugin will do the rest. It will install all "must have" plugins just in one click.



== Installation ==


1. Upload `dwordpress-perfection` to the `/wp-content/plugins/`  directory
2. Activate the plugin through the 'Plugins' menu in WordPress
